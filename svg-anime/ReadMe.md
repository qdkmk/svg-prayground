# vivus-map
vivus.jsとsnap.svg.jsを利用。
svgのイラスト(街)と地図をゼロから描画するアニメーション。
vivus.jsはスクロール発火もやってくれるそう。
描画をごりごり書かかなくてすむ。まるっとおまかせできるから楽そう。
vivusで描画、snap.svgは地図描画後にピンを下ろす動作を担っている。


vivusの設定についてはligの記事がわかりやすい
